# 🚀 Desafios de Lógica de Programação - JavaScript

Este repositório contém uma coleção de algoritmos desenvolvidos para resolver problemas de tomada de decisão, cálculos financeiros e regras de negócio utilizando **JavaScript**.

---

## 📂 Desafios Incluídos

### 1. 🏎️ Concessionária de Veículos
Algoritmo para cálculo de venda de automóveis com diferentes modalidades de pagamento.
* **PIX:** 15% de desconto.
* **Crédito:** 5% de juros (até 24x).
* **Financiamento:** 20% de juros (até 48x).

### 2. ☕ Cafeteria Gourmet
Sistema de fidelização e brindes.
* Compras > R$ 50,00 ativam a verificação de fidelidade.
* **Com Cartão:** 10% de desconto.
* **Sem Cartão:** Brinde especial (Cookie).

### 3. 📦 Frete E-commerce
Cálculo logístico baseado em região e valor de compra.
* **Sudeste:** R$ 10,00 | **Sul:** R$ 20,00 | **Outras:** R$ 30,00.
* **Regra de Ouro:** Compras acima de R$ 250,00 ganham **Frete Grátis**.

### 4. 🏦 Simulador de Empréstimo
Análise de risco de crédito baseada em margem consignável.
* **Regra:** A parcela não pode exceder 30% do salário bruto do cliente.

### 5. 🎟️ Bilheteria de Cinema
Lógica de aplicação de meia-entrada automática.
* **Critérios:** Menores de 12 anos, maiores de 60 anos ou estudantes pagam 50% do valor (R$ 20,00).

---

## 🛠️ Tecnologias e Conceitos Aplicados
* **JavaScript (ES6+)**
* **Estruturas Condicionais:** `if`, `else if`, `else`.
* **Operadores Lógicos:** `||` (OU) e `&&` (E).
* **Interação com Usuário:** `prompt()`, `alert()`.
* **Manipulação Numérica:** `toFixed(2)` para formatação de moeda.

| Desafio | Condição Principal | Resultado Esperado |
